El objetivo de este proyecto es crear una página web que permita a los usuarios conocer la oferta de servicios de la empresa, así como los productos que se ofrecen. Además, se busca que los usuarios puedan contactar con la empresa de manera sencilla y rápida. Por último, se busca que la página web sea atractiva y fácil de navegar para los usuarios. Nuestros clientes objetivo se basan en aquellos que tienen un interes en el mundo gastronomico en general, apuntando a formar una clientela solida que sea capaz de ver que somos, y demostrarles la mas alta calidad gastronomica que se puede ofrecer. Y nosotros tenemos la confianza de que nos ganaremos su confianza ya que contamos con un grupo sumamente capacitado para poder cumplir sus expectativas. Las palabras clave para este proyecto son: servicios productos contacto empresa cenas reservaciones tipos de comidas platos regionales platos internacionales menú ubicación Crear una rama: git branch branchName Movernos a otra rama: git checkout branchName Crear una rama en el repositorio local: git branch nombre-de-la-rama o git checkout -b nombre-de-la-rama. Publicar una rama local al repositorio remoto: git push origin nombre-de-la-rama.
